() chainable

Inherited from ya.Core: src/core.js:53

Initialize getters and setters for each property passed in config object. To track changes assign an event on for ex. propertyChange

Example:

var obj = ya.Core.$create({ config : { value : true } });

obj.addEventListener( 'valueChange', function () { alert('property changed!') } );
